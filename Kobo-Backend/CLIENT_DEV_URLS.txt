Kobo stack — API base URL for local Laravel

  Start the API so phones on your Wi-Fi can reach it:
    php artisan serve --host=0.0.0.0
  (Default `php artisan serve` binds to 127.0.0.1 only — mobile devices cannot connect.)

Kobo-Frontend (Next.js)
  Set NEXT_PUBLIC_API_URL in .env.local to your Laravel origin.

Kobo-Mobile (Expo)
  Set EXPO_PUBLIC_API_URL in app.config.js (extra) or .env as supported by your Expo setup.

  - Web / same machine: http://127.0.0.1:8000
  - Android emulator: http://10.0.2.2:8000
  - iOS simulator: http://127.0.0.1:8000
  - Physical device: http://<your-PC-LAN-IP>:8000 (PC and phone on same Wi-Fi; allow port 8000 in Windows Firewall)

Sanctum: both apps use Authorization: Bearer <token> from POST /api/auth/login.
